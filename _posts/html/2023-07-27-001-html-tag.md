# `<html></html>` 태그

- 카테고리 : None
- 사용 부분 : 문서의 최상단 부분 입니다.
- 내부 요소 : `<head></head>`요소가 먼저 들어오고 그 뒤에 `<body></body>`요소가 들어갑니다.
- 속성 : 
  - `Global 속성`, 
  - `manifest`
