# `<blockquote></blockquote>` 태그

- 카테고리 : 
  - `Flow content`
  - `Sectioning content`
  - `formatBlock candidate`
- 사용 부분 : 
  - `Flow content`가 기대되는 모든 문맥에서 사용할 수 있습니다.
- 내부 요소 : 
  - `Flow content`의 요소를 넣을 수 있습니다.
- 속성 : 
  - `Global 속성`
  - `cite` : 해당 요소가 인용한 소스의 주소를 지정합니다.
