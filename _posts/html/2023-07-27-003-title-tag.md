"title" 요소는 HTML 문서의 제목을 나타내는 메타데이터 요소입니다. 

---

# `<title></title>` 태그

- 카테고리 : `Metadata content`
- 사용 부분 : 
  - html 태그의 첫번째 자식 태그 입니다. 
  - 다른 요소를 포함할 수 없습니다. 
  - 또한 `<head></head>`에서 1번만 사용되어야 합니다.
- 내부 요소 : Text(글) 
- 속성 : 
  - `Global 속성`
