`<hr>` 요소는 수평 줄을 나타내는 HTML 태그로, 컨텐츠에서 주제의 변화나 새로운 섹션의 시작을 시각적으로 나타내는 역할을 합니다.

---

# `<hr>` 태그

- 카테고리 : 
  - `Flow content`
- 사용 부분 : 
  - `Flow content`가 기대되는 모든 문맥에서 사용할 수 있습니다.
- 내부 요소 : Empty
- 속성 : 
  - `Global 속성`
