li 태그는 목록의 항목을 나타내는 요소입니다. 부모 요소로 ol, ul, 또는 menu 요소를 가지고 있을 경우 해당 부모 요소의 목록 항목으로 취급됩니다.

---

# `<li></li>` 태그

- 카테고리 : None
  
- 사용 부분 : 
  - `<ol>` tag
  - `<ul>` tag
  - `<menu>` tag
  
- 내부 요소 : 내부요소가 없거나 `<li>` 태그를 포함하고 있습니다.

- 속성 : 
  - `Global 속성`
  - `<ol>` tag에 포함되어 있으면 `value` 속성을 가진다.
