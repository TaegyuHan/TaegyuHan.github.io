"base" 요소는 HTML 문서의 메타데이터 요소로서, 웹 페이지의 기본 URL을 설정하는데 사용됩니다.

---

# `<base>` 태그

- 카테고리 : `Metadata content`
- 사용 부분 : `<head></head>` 내에서 사용 가능합니다.
- 내부 요소 : Empty 아무것도 내부에 포함하지 않습니다.
- 속성 : 
  - `Global 속성`
  - `href` : 기본 url 저장
  - `target` : 링크를 열 때 연결된 리소스가 어떤 방식으로 열릴지를 정의합니다.
