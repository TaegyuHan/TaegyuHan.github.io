# Markdown 란?

> **( 출처 - 위키백과 )** <br>
마크다운(markdown)은 일반 텍스트 기반의 경량 마크업 언어다. 일반 텍스트로 서식이 있는 문서를 작성하는 데 사용되며, 일반 마크업 언어에 비해 문법이 쉽고 간단한 것이 특징이다. HTML과 리치 텍스트(RTF) 등 서식 문서로 쉽게 변환되기 때문에 응용 소프트웨어와 함께 배포되는 README 파일이나 온라인 게시물 등에 많이 사용된다.

* * *

# 제목 사용법 (Header)
주제목과 소제목을 작성합니다.
```markdown
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6
```

* * *

# 강조 (Emphasis)
글자를 강조하고 싶을 떄 사용합니다.
```markdown
*이텔릭체*
**두껍게**
```

* * *

# 목록 (List)
목록을 나타낼 때 사용합니다.
```markdown
1. 순서 리스트
1. 순서 리스트

- 리스트
  - 리스트

- 대쉬 (사용가능)
* 별표 (사용가능)
+ 더하기 (사용가능)
```

1. 순서 리스트
1. 순서 리스트

- 리스트
  - 리스트

- 대쉬 (사용가능)
* 별표 (사용가능)
+ 더하기 (사용가능)

# 링크 (Links)
원하는 링크를 연결할 때 사용합니다.
```markdown

[NAVER](https://naver.com "링크를 설명하는 문구를 작성해주세요.")

[현재 파일의 상대적 위치 링크](../users/login)
```

# 이미지 (Image)
이미지를 표현할 때 사용합니다.
```markdown
![대체 텍스트(alternative text)를 입력하세요!](http://www.gstatic.com/webp/gallery/1.jpg "링크 설명(title)을 작성하세요.")

![Kayak][logo]

[logo]: http://www.gstatic.com/webp/gallery/2.jpg "To go kayaking."
```

![대체 텍스트(alternative text)를 입력하세요!](http://www.gstatic.com/webp/gallery/1.jpg "링크 설명(title)을 작성하세요.")

# 이미지링크 (Image)
이미지를 클릭했을 때 원하는 링크로 이동하게 해줍니다.
```markdown
[![favicon](/img/favicon/favicon.svg)](https://github.com/TaegyuHan)
```

# 코드 (Code)
( ` ) 문자를 3번 입력후 자신이 원하는 코드 종류를적어 code 박스를 만듭니다.
```python

a = 'hello my blog!'
print(a)
>>> hello my blog!

```

* * *

# 글자 코드 (Text code)

```markdown

```

# 인용문 (BlockQuote)
```markdown
> 인용문

>> 1번 중첩 인용문
>>> 2번 중첩 인용문
>>> 3번 중첩 인용문

```

> 인용문
>> 1번 중첩 인용문
>>> 2번 중첩 인용문
